# SWADE Tools

A FoundryVTT module providing automation and quality-of-life improvements for the Savage Worlds Adventure Edition (SWADE) game system.

## Core Features

- Combat automation: automatic hit/raise detection, damage calculations, soak rolls, and status effect management
- Situational modifiers: distance, gang up, ranged in melee, scale, cover
- Character management: auto-update wounds/tokens based on wildcard status and size
- Simplified UI: streamlined roll dialogs and chat messages for easier gameplay
- Macro support: boost/lower trait, attack macros, batch rolling for selected tokens
- Multi-language support: English, French, Portuguese, Spanish, Italian, German, Catalan, Chinese

## Target Users

Game Masters and players using FoundryVTT with the SWADE system who want automated combat mechanics and reduced bookkeeping.
